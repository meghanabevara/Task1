# Titanic Data Preprocessing

## 📌 Project Overview

This project performs data preprocessing on the Titanic dataset.
The goal is to clean the data and prepare it for further analysis or machine learning.

---

## 📂 Dataset

* **titanic.csv** → Original dataset
* **cleaned_titanic.csv** → Processed dataset after cleaning

---

## ⚙️ Steps Performed

* Handling missing values
* Filling missing values using mean and mode
* Encoding categorical variables (e.g., Gender)
* Data cleaning and preparation

---

## 🛠️ Tools Used

* Python
* Pandas
* Jupyter Notebook

---

## 📁 Project Structure

task-1-data-preprocessing/
│
├── titanic.csv
├── cleaned_titanic.csv

---

## ▶️ How to Run

1. Open `preprocessing.ipynb`
2. Run all cells step-by-step
3. The cleaned dataset will be saved as `cleaned_titanic.csv`

---

## ✅ Output

A cleaned dataset ready for analysis and machine learning.
